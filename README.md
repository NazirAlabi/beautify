# Beautify by Ramat — Finance Tracker

An elegant, premium "liquid glass" financial tracking dashboard custom-built for **Beautify by Ramat**.

## Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4 (Glassmorphism & animated background gradient mesh)
- **Icons**: Lucide React
- **Charts**: Recharts
- **State Management**: React Context (`DataContext`, `ThemeContext`) with local storage persistence

## Key Features
- **Transaction Logs**: Record income (with multiple service selections) and expenses.
- **Dynamic Client Management**: Add, search, edit, or delete customer details. Inline client profile creation inside the transaction flow.
- **Financial Analytics**: Clean insight summaries, service breakdown progress bars, and profit margins.
- **Adaptive Theme**: Dynamic theme switching (Light / Dark mode).
- **Ghanaian Cedi (₵)** support natively integrated.

## Getting Started

### Development
```bash
npm install
npm run dev
```

### Production Build & Deploy
```bash
npm run build
npm run ship
```
